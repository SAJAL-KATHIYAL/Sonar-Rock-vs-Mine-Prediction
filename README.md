# Sonar Rock vs Mine Prediction

This project uses Machine Learning to classify sonar signals as either Rock or Mine.

## Algorithm Used
Logistic Regression

## Libraries Used
- NumPy
- Pandas
- Scikit-learn

## Steps
1. Data Collection
2. Data Preprocessing
3. Train-Test Split
4. Model Training using Logistic Regression
5. Model Evaluation using Accuracy Score
6. Prediction System

## Dataset
Sonar dataset containing 60 features representing sonar signals.
